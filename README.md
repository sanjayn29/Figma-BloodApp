# 🩸 Blood Drop – UI/UX Design

**Blood Drop** is a **UI/UX design project** created to visualize a platform that connects **blood donors with people in need of blood**.  
The project focuses on designing a **clean, user-friendly mobile and web interface** that simplifies the process of requesting and donating blood.

This project was designed entirely using **Figma**, focusing on modern UI principles, accessibility, and intuitive user experience.

---

# 🎯 Project Objective

The main goal of this design is to create an interface that allows users to:

- Quickly find blood donors
- Request blood in emergency situations
- Register as a blood donor
- Track donation requests
- Improve communication between donors and recipients

The design prioritizes **simplicity, clarity, and accessibility**, which are essential for emergency healthcare applications.
